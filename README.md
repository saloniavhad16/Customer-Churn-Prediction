# Customer Churn Prediction Platform

## 📌 Project Overview
This project predicts whether a customer is likely to leave (churn) or stay using Machine Learning.

## 🚀 Features
- Customer Churn Prediction
- XGBoost Machine Learning Model
- Streamlit Web Application
- Interactive User Input
- Accuracy around 80%

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- XGBoost
- Streamlit
- Git
- GitHub

## 📂 Dataset
Telco Customer Churn Dataset

## ▶️ How to Run

1. Install dependencies

```bash
pip install -r requirements.txt
```

2. Train the model

```bash
python train.py
```

3. Run Streamlit App

```bash
streamlit run app.py
```

## 📈 Output
- Customer is likely to stay
- Customer is likely to leave
